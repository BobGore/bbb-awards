Google Sheets Setup

1. Create a new Google Sheet with columns: timestamp, voter, award_id, nominee

2. In your Sheet, go to Extensions → Apps Script

3. Paste the Apps Script code (copy from below), click Deploy → New Deployment → Web App

4. Set access to "Anyone", copy the Web App URL, paste it above

5. Click "Test Connection" to confirm it's working
